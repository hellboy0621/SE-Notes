## 设置
### 外网访问
```
# 打开配置文件把下面对应的注释掉
# bind 127.0.0.1 
# Redis默认不是以守护进程的方式运行，可以通过该配置项修改，使用yes启用守护进程，设置为no
daemonize no
# 保护模式
protected-mode no 
```
